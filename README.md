### A Note From Present Bobby (27)

I created this repository because I was tired of living a small life.

I used to let people use me.  
I used to be afraid of saying no.  
I used to think my only value was being useful.

This repo is proof that I chose to become better.

When I die, do not cry too much.  
Instead, read this repository.  
It contains the best and worst parts of me — completely unfiltered.

Thank you for being part of my story.

— Bobby
